# STUDENT MANAGEMENT SYSTEM
A Student Management System project implemented using Java and MySQL. It allows users to perform CRUD (Create, Read, Update, Delete) operations on student data such as 
1) Roll Number
2) First Name 
3) Last Name 
4) Major 
5) Phone Number
6) GPA 
7) Date of Birth

# DATABASE COLUMNS
1) Roll Number -> roll_no INT PRIMARY KEY 
2) First Name -> first_name VARCHAR(50)
3) Last Name -> last_name VARCHAR(50)
4) Major -> course_major VARCHAR(20)
5) Phone Number -> phone_no VARCHAR(15)
6) GPA -> gpa DECIMAL(4,2)
7) Date of Birth -> dob DATE
